Feedback Form Script Wizard Demo Site

Introduction

This site is merely a demo site for the contact form script created by thesitewizard.com's Feedback Form Script Wizard. It is intended to show how the feedback form script will work when used on a real website.

To check out the demo, please proceed to the Feedback Form Script Demo Page.
